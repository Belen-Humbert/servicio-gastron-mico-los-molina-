# 🔥 Servicio Los Molina — Landing Page

Landing page oficial del **Servicio Gastronómico Los Molina**, empresa familiar mendocina con más de 15 años llevando la gastronomía argentina a eventos de toda la región.

## 🌐 Demo

[Ver sitio en vivo](https://belen-humbert.github.io/servicio-gastron-mico-los-molina-/)

## 📋 Descripción

Sitio web de una página (landing page) para el servicio gastronómico familiar Los Molina, con información sobre sus servicios, menú, equipamiento y datos de contacto.

## 🍽️ Servicios que ofrece

- Parrilla a leña y asado completo
- Empanadas caseras
- Pernil al horno
- Picadas criollas
- Guisos y locro de temporada
- Servicio de mozos
- Banquete completo (tablones, sillas, mantelería, vajilla)

## 🗺️ Zona de cobertura

Lavalle · Las Heras · Mendoza Capital · Guaymallén · Godoy Cruz · San Martín

## 🛠️ Tecnologías

- HTML5
- CSS3 (sin frameworks)
- JavaScript vanilla
